# 🍔 B7Burger

## 🚀 Sobre o Projeto

O **B7Burger** é um projeto de interface web desenvolvido para simular a experiência de uma hamburgueria online moderna. O objetivo principal foi criar uma página visualmente atrativa, responsiva e intuitiva, proporcionando uma navegação agradável para o usuário.

O projeto foi construído com foco em:

* ✅ Estrutura semântica em HTML5
* ✅ Estilização moderna com CSS3
* ✅ Layout responsivo
* ✅ Organização de arquivos e componentes
* ✅ Experiência visual agradável
* ✅ Boas práticas de desenvolvimento Front-End

Além do visual moderno, o projeto demonstra atenção à organização do código e à experiência do usuário — características valorizadas por recrutadores e equipes de desenvolvimento.

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de praticar e aprimorar habilidades em desenvolvimento Front-End, especialmente:

* Estruturação semântica com HTML5
* Criação de layouts modernos
* Estilização com CSS3
* Responsividade
* Organização de arquivos
* Construção de interfaces voltadas para experiência do usuário

Também serve como peça de portfólio para demonstrar domínio dos fundamentos essenciais do desenvolvimento web.
